处理字符串里有特殊字符的时候：注意加“[]”括号处理

1.彻底清理浏览器缓存：

清理火狐，谷歌的浏览器缓存————可以从 safel 浏览器里进行彻底清除。

2.mac 查询文件

which solcjs

3.mac 下设置vi/vim 高亮

步骤1：

cp /usr/share/vim/vimrc ~/.vimrc

先复制一份vim配置模板到个人目录下

注：redhat 改成 cp /etc/vimrc ~/.vimrc

步骤2：

vi ~/.vimrc

进入insert模式，在最后加二行

syntax on

set nu!

保存收工。

4.mac 安装指定版本软件：

brew search go

brew install go@1.6

echo ‘export PATH=”/usr/local/opt/go@1.6/bin:$PATH”‘ >> ~/.bash_profile

source ~/.bash_profile

4.mac 更新 brew的源

cd “$(brew –repo)/Library/Taps/homebrew/homebrew-core” git remote set-url origin git://mirrors.ustc.edu.cn/homebrew-core.git echo ‘export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles‘ >> ~/
.bash_profile
source
~/.bash_profile
brew update

5.mac 下查看ip地址：
ifconfig | grep “inet “ | grep -v 127.0.0.1

6.软件下载：

如果 在苹果商店下载不到的，可以在软件官网进行下载。**
