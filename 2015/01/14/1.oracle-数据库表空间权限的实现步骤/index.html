1.Oracle 数据库表空间权限的实现步骤

By Zhaofa Yin

2015-01-14 更新日期:2018-01-22

文章目錄

1.创建数据库实例，通过工具 database configuration assistant .
set oracle_sid=ORCL 修改当前的服务名；
然后在plsql登录 用户名 sysdba 密码 sysdba 服务名为新建的那个库的。连接为 SYSDBA
2
create temporary tablespace huasheng
tempfile ‘D:\oracle\product\10.2.0\db_1\oradata\Oracle10\huasheng.dbf’
size 50m
autoextend on
next 50m maxsize 20480m
extent management local;

3
create tablespace huasheng
logging
datafile ‘D:\oracle\product\10.2.0\db_1\oradata\Oracle10\huasheng.dbf’
size 50m
autoextend on
next 50m maxsize 20480m
extent management local;

4
create user wozywei identified by wozywei
default tablespace user_data
temporary tablespace user_temp;

5
grant connect,resource,dba to wozywei;

/分为四步 /
/第1步：创建临时表空间 /
create temporary tablespace user_temp
tempfile ‘D:\oracle\oradata\Oracle9i\user_temp.dbf’
size 50m
autoextend on
next 50m maxsize 20480m
extent management local;

/第2步：创建数据表空间 /
create tablespace user_data
logging
datafile ‘D:\oracle\oradata\Oracle9i\user_data.dbf’
size 50m
autoextend on
next 50m maxsize 20480m
extent management local;

/第3步：创建用户并指定表空间 /
create user username identified by password
default tablespace user_data
temporary tablespace user_temp;

/第4步：给用户授予权限 /
grant connect,resource,dba to username;
grant connect,resource,dba to WOEGOLD
