1.Rabbix-docker 化搭建

By Zhaofa Yin

2015-01-14 更新日期:2018-01-22

文章目錄

启用容器方法1，会多一个镜像
docker run -d –hostname my-rabbit –name sde-rabbit -p 8080:15672 rabbitmq:3-management

启用容器方法2，需要进入容器，启用一个命令（下面有rabbitmq_management）
docker run -d –hostname my-rabbit –name sde-rabbit -p 8080:15672 rabbitmq:latest

启用容器方法3（自己的做的镜像，安装了一些插件，无需再次下载）
docker run -d –hostname my-rabbit –name sde-rabbit -p 8080:15672 sde-rabbitmq/yinzf:latest

运行插件：
docker run -d –hostname my-rabbit –name sde-rabbit -p 8080:15672 rabbitmq:latest

启用插件：
rabbitmq-plugins enable rabbitmq_mqtt

进入容器：
docker exec -it cc941058e34a /bin/bash

启用此插件(不启用不能访问 mq的web后台)
./rabbitmq-plugins enable rabbitmq_management

查看安装的软件：
cd /usr/lib/rabbitmq/bin
./rabbitmq-plugins list

停用rabbit服务(docker stop 2083bed28adc)
./rabbitmqctl stop

启用rabbit服务(docker start 2083bed28adc)
./rabbitmq-server -detached

启用此插件
./rabbitmq-plugins enable rabbitmq_mqtt

如果要禁用用此命令即可
./rabbitmq-plugins disable rabbitmq_mqtt
查看已经安装的插件

==创建本地镜像操作：
docker commit 1318320e8326 rabbitmq_web_mqtt/yinzhaofa

==运行本地镜像：
docker run -d –hostname my-rabbit –name sde-rabbitmq_web_mqtt -p 61613:61613 -p 18883:8883 -p 11883:1883 -p 15671:5671 -p 35672:5672 -p 15672:15672 -p 15675:15675 -p 14369:4369 -p 25672:25672 rabbitmq_web_mqtt/yinzhaofa:latest
