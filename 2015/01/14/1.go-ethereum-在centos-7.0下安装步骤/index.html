上代码

执行一下步骤：

yum install golang
yum install gmp-devel
git clone https://github.com/ethereum/go-ethereum
cd go-ethereum
make geth
build/bin/geth

如果是新机器，可能遇到一下问题：

1.需要安装git：

yum install git

git –version

yum remove git

2.下载代码可能遇到一下问题：需要如下操作：

需要通过github的用户生成 公私钥，公钥上传到 git—seting——ssh key

执行：

ssh-keygen -t rsa -C”mail@mail.com”

cat ~/.ssh/id_rsa.pub

测试：

ssh -T git@github.com来验证是否成功。如果出现象：hi xxx. You’ve successfully authenticated, but GitHub does not provide shell access.则说明连接成功。

3.执行 下载

git clone https://github.com/ethereum/go-ethereum.git

如果报错：

Github git clone error: Peer’s Certificate has expired

解决方法：

[cos@localhost github]$ date
Wed Aug 26 04:07:42 EDT 2015 [cos@localhost scripts]$ sudo yum -y install ntp
[cos@localhost scripts]$ sudo chkconfig ntpd on
Note: Forwarding request to ‘systemctl enable ntpd.service’.
ln -s ‘/usr/lib/systemd/system/ntpd.service’ ‘/etc/systemd/system/multi-user.target.wants/ntpd.service’ [cos@localhost scripts]$ sudo ntpdate time.apple.com12 May 07:52:16 ntpdate[28563]: step time server 17.253.84.253 offset 22477005.642802 sec

以上通过后，在执行：

git clone https://github.com/ethereum/go-ethereum.git

4.执行 make geth 可能报错需要 在go1.7下执行

首先删除之前的go

yum remove go

安装go1.7

参考 ：（同目录下“2. centos 7.0 安装go 1.7”）

或者链接：（如果还有此链接）https://www.howtoing.com/how-to-install-go-1-7-on-centos-7/
