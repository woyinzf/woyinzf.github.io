2.rabbitmq常用命令

By Zhaofa Yin

2015-01-14 更新日期:2018-01-22

文章目录

启动服务
/usr/lib/rabbitmq/bin/rabbitmq-server start &
Ø 停止服务
/usr/lib/rabbitmq/bin/rabbitmq-server stop &
Ø 查看队列中的数据(**)
/usr/lib/rabbitmq/bin/rabbitmqctl list_queues
Ø 查看节点状态(**)
/usr/lib/rabbitmq/bin/rabbitmqctl status
Ø 查看分布式节点状态
/usr/lib/rabbitmq/bin/rabbitmqctl cluster_status
Ø 停止应用
/usr/lib/rabbitmq/bin/rabbitmqctl stop_app
Ø 启动应用
/usr/lib/rabbitmq/bin/rabbitmqctl start_app
Ø 重设应用
/usr/lib/rabbitmq/bin/rabbitmqctl reset
Ø 加入分布式节点
/usr/lib/rabbitmq/bin/rabbitmqctl cluster 节点标识
Ø 具体参考:
/usr/lib/rabbitmq/bin/rabbitmqctl –help
Ø 查看未确认消息：(**)
/usr/lib/rabbitmq/bin/rabbitmqctl list_queues -p / name
Ø 看到两个临时队列的名称
/usr/lib/rabbitmq/bin/rabbitmqctl list_bindings
Ø 看交换器
/usr/lib/rabbitmq/bin/rabbitmqctl list_exchanges
Ø 查看用户列表
/usr/lib/rabbitmq/bin/rabbitmqctl list_users

监控容器上跑着的日志
docker logs -ft cfb511ab4e45

查看近期10条容器内应用的日志
docker logs –tail -100ft cfb511ab4e45
